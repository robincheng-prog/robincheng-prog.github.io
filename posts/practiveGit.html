作者： Ayden

1. Git 是什麼？

Git 是一種「版本控制系統（Version Control System, VCS）」，主要用途：

  • 記錄程式碼修改歷史
  • 多人協作開發
  • 回復舊版本
  • 管理不同功能分支
  • 搭配 GitHub / GitLab 做遠端備份

簡單理解：

Git 就像程式碼的「時間機器」。
