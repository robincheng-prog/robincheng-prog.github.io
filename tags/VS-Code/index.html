Visual Studio(VS) Code using NXP MCUXpresso plugin

1. Install VS code, download location:

https://code.visualstudio.com/download, select development environment for yours .

2. Install VS Code.

(Current VS code is code_1.94.2-1728494015_amd64.deb)

3. Install the NXP MCUXpresso plugin after installed VS Code :

Please refer below link

https://marketplace.visualstudio.com/items?itemName=NXPSemiconductors.mcuxpresso

Follow the NXP MCUXpresso plugin guideline:

1. Launch VS Code Quick Open (Ctrl+P), paste the following command, and press enter.

PS: Standalone Tools are not needed and can be omitted.

2. MCUXpresso Configuration Tools are available on the NXP website.

Download and install. https://www.nxp.com/design/design-center/software/development-software/mcuxpresso-software-and-tools-/mcuxpresso-config-tools-pins-clocks-and-peripherals:MCUXpresso-Config-Tools

3. If there is an error, you need to install Python, Ctrl+Shift+X, enter python, and install Python

Install 7z https://www.7-zip.org/download.html

4. Install related tools

Dependency tools and software components can be installed using MCUXpresso Installer
https://mcuxpresso.nxp.com/mcux-vscode/latest/html/MCUXpresso-Installer.html

![MCUXpresso Installer](images/MCUXpresso Installer.png)

Download MCUXpressoInstaller.deb.bin for Linux,

When installing again, a GUI screen will appear and every related tool must be installed, which will take a long time.
